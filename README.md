# southjordancanal
Website for the South Jordan Canal Company providing interactive canal maps, company information, and contact details.

## Features
- Interactive canal map with location services
- Company information and fee schedules
- Contact information for company officials
- Water delivery and season information

## Development
### Prerequisites
- Node.js
- npm

### Setup
1. Clone the repository
2. Install dependencies:
```
npm install
cd lambda && npm install
```

### Local Development
Run the development server:
```
npm start
```
The site will be available at http://localhost:9000

### Build
Build the project:
```
npm run build
```

## Deployment
The project is deployed using AWS Amplify and includes both frontend and Lambda function deployments.
